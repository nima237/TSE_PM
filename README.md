## راه‌اندازی Git و Push

برای push کردن روی GitHub، این دستورات رو در Terminal اجرا کن:

```bash
# 1. Initialize git repository
git init

# 2. Add all files
git add .

# 3. Commit
git commit -m "Initial commit: Professional Investment Market Presentation"

# 4. Add remote repository
git remote add origin https://github.com/nima237/TSE_PM.git

# 5. Push to main branch
git branch -M main
git push -u origin main
```
